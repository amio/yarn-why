# npm-why [![npm-version][npm-badge]][npm-link]

Identifies why a package has been installed.

## Install & Usage

```bash
npm install -g yarn-why
```

```bash
Usage

  $ yarn-why <package>

Examples

  $ yarn-why babel-core
```
